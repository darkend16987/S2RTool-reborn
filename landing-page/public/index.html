Cài đặt S2R Tool chỉ trong 5 phút với Docker Compose

1

Cài đặt Docker Desktop

Tải và cài Docker Desktop cho hệ điều hành của bạn. Sau khi cài, khởi động Docker Desktop và đảm bảo nó đang chạy (icon Docker xanh ở system tray).

open_in_new docker.com/products/docker-desktop
check_circle Xác nhận: Mở terminal, gõ docker --version — thấy version là OK
2

Lấy Gemini API Key

Truy cập Google AI Studio, đăng nhập bằng tài khoản Google, và tạo API Key mới. Copy key này lại — sẽ dùng ở bước 4.

open_in_new aistudio.google.com/app/apikey
lightbulb Free tier cho phép 60 requests/phút — đủ cho sử dụng cá nhân
3

Tải deployment package

Tải bản release mới nhất từ GitHub. Package bao gồm docker-compose.yaml, deploy.sh, và các file cấu hình cần thiết.

download Tải từ GitHub Releases
Hoặc clone bằng Git 
git clone https://github.com/darkend16987/S2RTool-reborn.git
cd S2RTool-reborn
4

Cấu hình environment

Tạo file .env trong thư mục project và điền Gemini API Key.

.env 
# === BẮT BUỘC ===
GEMINI_API_KEY=your_gemini_api_key_here

# === TÙY CHỌN ===
BACKEND_PORT=5001
FRONTEND_PORT=3001

# Docker Registry (không cần thay đổi)
DOCKER_REGISTRY=docker.io
DOCKER_USERNAME=kael16987
VERSION=latest

# Auto-update interval (seconds)
WATCHTOWER_INTERVAL=300
lightbulb Thay your_gemini_api_key_here bằng API key đã lấy ở bước 2
5

Khởi chạy S2R Tool

Chạy lệnh bên dưới để pull Docker images và khởi động toàn bộ hệ thống (backend + frontend + auto-updater).

Linux / macOS 
chmod +x deploy.sh
./deploy.sh

Deploy script sẽ tự động hỏi API key, cấu hình port, và khởi chạy.

Docker Compose 
# Pull images mới nhất
docker-compose -f docker-compose.production.yaml pull

# Khởi chạy (background mode)
docker-compose -f docker-compose.production.yaml up -d
check

Truy cập S2R Tool

Mở trình duyệt và truy cập:

http://localhost:3001
check_circle Nếu thấy trang chủ S2R Tool với các mode cards — cài đặt thành công!

Tự động (Watchtower)

Khuyến nghị

Nếu bạn dùng docker-compose.production.yaml, Watchtower đã được tích hợp sẵn. Nó tự động kiểm tra Docker Hub mỗi 5 phút và cập nhật khi có phiên bản mới.

info Không cần làm gì — update diễn ra hoàn toàn tự động
terminal

Thủ công (Manual)

Nếu không dùng Watchtower hoặc muốn update ngay lập tức:

Terminal 
# Di chuyển vào thư mục project
cd S2RTool-reborn

# Pull images mới nhất từ Docker Hub
docker-compose -f docker-compose.production.yaml pull

# Restart với images mới
docker-compose -f docker-compose.production.yaml up -d

# Kiểm tra logs
docker-compose -f docker-compose.production.yaml logs -f
lightbulb Dữ liệu (references, settings) được lưu trong Docker volumes — không bị mất khi update

error Docker Desktop không khởi động

Windows: Đảm bảo đã bật WSL 2 và Virtualization trong BIOS.

macOS: Đảm bảo đã cấp quyền trong System Preferences → Security & Privacy.

Kiểm tra Docker
docker info
docker-compose version
error Port 3001 đã bị chiếm

Thay đổi port trong file .env:

.env
FRONTEND_PORT=3002
BACKEND_PORT=5002

Sau đó restart: docker-compose -f docker-compose.production.yaml up -d

error API Key không hoạt động / lỗi 403

Kiểm tra:

  • API Key có đúng không (không có khoảng trắng thừa)
  • Tài khoản Google đã enable Gemini API chưa
  • Đã vượt quota chưa (kiểm tra ở Google AI Studio)
error Render chậm hoặc timeout

Gemini API có thể chậm trong giờ cao điểm. App có timeout 300 giây cho mỗi render request. Nếu timeout thường xuyên:

  • Thử lại sau vài phút
  • Giảm kích thước ảnh input
  • Kiểm tra kết nối internet
